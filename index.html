web-application-vulnerability-scanner

This is a Python-based web vulnerability scanner designed to identify common web application vulnerabilities, including XSS, SQL Injection, CSRF, and more. The scanner uses the requests library for making HTTP requests and BeautifulSoup for parsing HTML forms

Features

  • XSS (Cross-Site Scripting) Detection: Scans for potential XSS vulnerabilities using predefined payloads.
  • SQL Injection Detection: Identifies potential SQL injection vulnerabilities using typical SQLi payloads.
  • CSRF (Cross-Site Request Forgery) Detection: Checks for CSRF vulnerabilities by analyzing the response.
  • Open Redirect Detection: Tests for open redirects which could be exploited for phishing attackse.
  • Cookie Security Check: Verifies that cookies are set with secure and HttpOnly flags.

IP Scanner

This is a simple IP scanner implemented in Python 3, allowing you to scan a subnet to find active IP addresses using ICMP ping requests..

Features

  • Scans a specified subnet for active IP addresses.
  • Utilizes Python's subprocess module to send ICMP echo requests and check responses.
  • Displays the list of active IP addresses found during the scan.

Network-Sniffer-and-Packet-Analyzer-CLI-Tool

This Python script is a powerful network sniffer and packet analyzer . It captures live network traffic on a specified interface, identifies potential threats based on user-defined keywords, and alerts the user. The tool is built with the Scapy library, providing a flexible and detailed network analysis

nmap-scanner

This script provides an advanced interface for performing Nmap scans. It supports a variety of scan types, including TCP SYN, TCP Connect, UDP, OS detection, version detection, and aggressive scanning. It also includes features for custom port ranges, script scanning, and saving scan results in multiple formats.
